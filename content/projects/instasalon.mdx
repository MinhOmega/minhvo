## Overview

A comprehensive appointment management system designed for nail salons in the US. The system streamlines scheduling and management for salon managers, technicians, and customers.

## Key Features

- Appointment scheduling system
- Staff management
- Customer booking interface
- Payment processing / In-app purchase
- Real-time notifications / chat
- Analytics integration
- Offline capabilities

## Technical Stack

### Development
- TypeScript
- React Native
- Redux
- Firebase Services
- Styled-components

### Firebase Integration
- App Distribution
- Cloud Firestore
- Cloud Messaging
- Analytics (GTM, GA4)
- Crashlytics

## Implementation Details

- Custom payment method integration
- Performance optimizations
- Real-time data synchronization
- Push notification system
- Analytics implementation

## Role & Responsibilities

- Implemented payment systems
- Optimized app performance
- Integrated Firebase services
- Implemented analytics
- Managed crash reporting
