## Overview

A comprehensive e-commerce and sales management platform designed specifically for Small and Medium-sized Businesses (SMBs). Built with modern technologies and focusing on server-side rendering for optimal performance.

## Key Features

- E-commerce platform
- Sales management system
- Server-side rendering
- Static page generation
- Responsive UI with animations
- Form validation system

## Technical Stack

### Frontend
- Next.js 14 platform
- TailwindCSS for styling
- TypeScript for type safety
- Form validation libraries

### Backend & Infrastructure
- MongoDB for database
- Prisma ORM
- Docker containerization
- Server-side rendering (SSR)
- Static site generation (SSG)
